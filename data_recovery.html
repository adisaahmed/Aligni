Database Backup and Disaster Recovery

File Based Backups -

Are retained on disk for 4 weeks and then backed up to an Azure Recovery Service Vault daily and retained for 7 days, Sundays backups are retained for 4 weeks, the last daily backup of each month is retained for 12 months, last daily backup of the year is retained for 7 years.

SQL Backups

  • Full Backups are done on Sunday night to a network share
  • Differential Backups are done nightly Monday through Saturday to a network share
  • Transaction Log Backups are done hourly to a network share

File Backups

  • Full Backups are done on Sunday night to a network share
  • Differential Backups are done nightly Monday through Saturday to a network share

Machine Based Backups -

Are backed up to an Azure Recovery Service Vault daily and retained for 14 days, Saturdays backups are retained for 4 weeks and the last Saturday backup of each month is retained for 2 months.

All backups are stored on Geo-Redundant storage and the current DR plan would be to retrieve the last backups from the replicated storage in the East Coast data center and bring the servers up there. Microsoft has in Public Preview a fully automated DR solution that TMC will be employing once it’s has been fully released to the public.
