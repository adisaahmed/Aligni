Easy Start 1

Microsoft Dynamics 365

Easy Step 1

Economy Implementation Package

Starting at ₦6,000

Learn More / Contact Us

This implementation package is for the organization that just requires set up, implementation services and training for the Financial Modules.

  1. Establish new Dynamics 365 Business Central environment.
    • Set up a maximum of 10 users with multi-factor authentication with access to each database.
    • Set up on production environment
    • Set up on Sandbox environment
    • Configure upgrade times and notifications for each instance
  2. Show customer how to set up one user in Business Central environment (One Production and One Sandbox) with permissions provided by customer. Customer will set up all users.
  3. Assist client with launching start up configuration guide to set up Company and related information. Customer will go through these screens with help from iCepts as needed.
  4. Show client how to set up necessary posting groups.
  5. Provide Blank Import Templates for Data Migration from other systems for Vendor, Customer, and Chart of Accounts Master Records. Customer is responsible for populating these and for the import data. Aligni will provide instruction on how to import. Customer will do the actual import of data.
  6. Aligni will review with client areas of reporting and the concept of running reports.
  7. Phone call to review go live status. Is the customer ready to go live?

Easy Start 2

Microsoft Dynamics 365

Easy Step 2

Business Implementation Package

Starting at ₦12,500

Learn More / Contact Us

This implementation Package includes everything in the Easy Start 1 Implementation Package as well as the following.

  1. Guide and assist in Set up of Sales order, invoice, and purchase order processing.
  2. Guide and assist in Set up of inventory items (basic set up only)
  3. Provide templates for Items to be imported. Client will populate items from template and import.
  4. Customer will do all testing on above to ensure they can process orders and create invoices.
  5. Show customer how to locate item reports and discuss month end closing using reports.
  6. Customer will provide all acceptance testing and notify iCepts when ready to go live.
